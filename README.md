# DevRadar Api

## Open Endpoints

Open endpoints require no Authentication.

* [Devs]: `POST /devs` -- Create new dev
* [Devs]: `GET /devs` -- List all devs
* [Devs]: `PUT /devs/:id` -- Update dev
* [Devs]: `DELETE /devs/:id` -- Delete dev
* [search]: `GET /search` -- Show dev by location
